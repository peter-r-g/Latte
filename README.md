# Latte
A work in progress 100% C# game engine.

## Features
* Renderer agnostic API.
  * Currently supports only Vulkan.

## Installation

### Requirements
* .NET 7.0
* Vulkan SDK

### Instructions
1. Clone the repository.
2. Launch through the `Latte.TestBed` project.
